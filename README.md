# Portfolio-Study

이 저장소는 학회 활동과 스터디에서 진행한 학습 및 실습 기록을 통합한 아카이브입니다.  
개별 프로젝트와는 별도로, **주간 학습 기록(TIL)**, **팀 학습 기록(Study)**, **실습형 미니 프로젝트**를 정리하여 학습 과정을 체계적으로 관리하고 있습니다.
